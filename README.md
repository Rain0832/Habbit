# Habbit
Habbit 兔效率管理软件

# 运行

### 运行环境

Windows Visio Studio 2022

### 必要依赖
- EasyX 图形库工具
- 软件字体 .ttf（项目内含）
- 软件背景 图片（项目内含）

### 运行操作
##### Habit_v5.0/
- 文件内有 `.sln` 文件，Window 环境下可以打开 VS2022，构建项目工程，完成更多操作
- 文件内还含有项目代码的完全文件

##### Release/
软件发行文件，内含 .exe 文件，可以直接运行软件
